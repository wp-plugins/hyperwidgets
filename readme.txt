An interface addition for Widgets to let you group your Sidebars, eg for each page template.

== Description ==

When you use Wordpress as a CMS, it's very common to create many sidebars, perhaps several per page template.
With this plugin you can group the sidebars with a name and exclude some.

[youtube http://www.youtube.com/watch?v=a_nzNYBQuIs]

== Installation ==

1. Upload `Hyperwidgets` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets page to enjoy

== Screenshots ==

1. In the select field you can find all your sidebars.
